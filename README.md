# EasyWordList

EasyWordList is a simple and customizable wordlist generator written in Python.

It allows you to generate all possible combinations of characters based on:
- lowercase letters
- uppercase letters
- numbers
- symbols

---

## Features

- Custom character set selection
- Configurable word length (up to a safe limit)
- No external libraries required
- Efficient counter-based generation
- Output saved directly to a file

---

## Build and Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/EasyWordList.git
   cd EasyWordList
   ```
2. Run:
   ```bash
   python easywordlist.py
   ```
